v6.7 - September 5, 2014

  • Redesigned tags. A color can now be assigned to each tag.
  • Added ability to backup and export data to Dropbox.
  • Added automatic backups option (Pro version only).
  • Added insulin on board information to home screen.
  • Integrated app with the DashClock widget.
  • Hungarian translations added thanks to Zoltán Mészáros!
  • New app icon.

v6.6.1 - July 27, 2014

  • Bug and crash fixes.

v6.6 - July 22, 2014

  • Added navigation drawer (swipe from edge of screen to open).
  • Added food database (user-managed, Pro version only).
  • Added option to backup/restore settings.
  • Added options to modify BG targets and bolus ratios from entry screen.
  • Android 2.2 no longer supported.
  • Bug fixes.

v6.5.5 - April 23, 2014

  • Added carbohydrates calculator to the edit entry screen.
  • Added ability to restore and import from Dropbox on Android 3.0+ (backing up to Dropbox will be added when it is supported by Dropbox).
  • Removed restrictions when setting starting and ending mealtimes.
  • Android 2.1 no longer supported.
  • Bug fixes.

v6.5.4 - March 5, 2014

  • Fix bolus calculation for mmol/L units.

v6.5.3 - March 4, 2014

  • Tags are now shown in entries list.
  • Added ability to filter entries and statistics by time.
  • Added BG target offset setting.
  • Made it easier to create reminders from entry screen.
  • Lots of small improvements.

v6.5.2 - January 28, 2014

  • Added ability to snooze reminders.
  • Added reminder options for vibrate and sound.
  • Updating entries no longer creates automatic reminders. Improved automatic reminders behavior to avoid multiple reminders being set at the same time.
  • Bug fixes.

v6.5.1 - January 15, 2014

  • Added home screen shortcuts for creating entries and reminders.
  • Fixed reminders for Pro version.
  • Bug fixes.

v6.5 - January 10, 2014

  • A1C is calculated and shown in statistics.
  • Added more filters. Added filtering by tags (Pro version only).
  • Added repeating reminders. Added automatic reminders (Pro version only).
  • Added option to use large numpad when inputting values.
  • Insulin on board warning now only appears for the current date's entries.

v6.4.4 - December 24, 2013

  • New date and time pickers.
  • Made it easier to create reminders.
  • UI refinements.
  • Fixed bugs and Android 4.4 issues.
  • Added French translations. Updated Italian translations.

v6.4.3 – September 30, 2013

  • Fixed bug when importing CSV data.

v6.4.2 – September 23, 2013

  • Added ability to import data using a CSV file.
  • Added setup guide for new users.
  • Enabled landscape mode.
  • Miscellaneous improvements, bug and crash fixes.

v6.4.1 – September 8, 2013

  • Added option to create entries that won’t be included in statistics.
  • Added ability to share entries.
  • Tap images to view them in full screen.
  • Performance improvements when loading data.
  • Lots of minor improvements.

v6.4 – August 22, 2013

  • Added tags functionality (all insulin types will be converted to tags automatically).
  • Added Google Analytics. This will help in crash and bug reporting (Requires internet permissions).
  • Bug fixes (Thanks to Fredrik for all his help).

v6.3.1 – July 20, 2013

  • Fixed editing date and time for exercise entries.
  • Removed home entries list.
  • Removed date and time format options. The app will automatically detect the correct formats.
  • Fixed crash for certain time zones.
  • Other bug fixes.

v6.3 – July 13, 2013

  • Added reminders (requires new permissions).
  • Fixed bug when saving exercise entries.
  • Fixed bug with incorrect time zones.

v6.2 - June 23, 2013

  • Updated and consistent UI for all Android versions.
  • Swipeable tabs!
  • Improved how negative boluses behave (negative boluses are disabled by default).
  • Performance improvements when editing entries.
  • Can choose any increment to round boluses by.
  • Fixed a bug when editing entries.

v6.1.1 - May 19, 2013

  • Fixed crash when tapping on statistics graph on certain devices.

v6.1 - May 19, 2013

  • Added option to color code low and high blood glucose.
  • Can now tap on the statistics graph to enlarge.
  • Added option to subtract from carbs boluses to adjust for low blood glucose.
  • Blood glucose targets and boluses will use the entry’s time instead of the device’s time.
  • Can now create entries with just notes.
  • Translated to Italian by Pierluigi Covelli!
  • Various bug fixes and improvements.

v6.0 - April 8, 2013

  • Improved picture quality.
  • Improved backup and restore. Multiple backup files can be made. You can choose which backup to restore from.
  • Fixed crash when trying to take a picture on a device without a camera.
  • Lots of UI fixes and various improvements.

v5.3 - March 8, 2013

  • New icon!
  • Updated some settings descriptions to make them more clear.

v5.2 - February 1, 2013

  • Fixed crash when viewing or creating entries with invalid times due to daylight savings.

v5.1 - November 26, 2012

  • Fixed Restore failure message to include name of backup file and its location.

v5.0 - November 25, 2012

  • Can now backup and restore data from Settings > Tools.
  • Added warnings when entering blood glucose values that are out of range. Can be enabled/disabled from Settings > Blood glucose
  • Added insulin on board warnings when giving a correction bolus when a bolus was already given. Can be enabled/disabled and configured from Settings > Bolus
  • Added dialogs to indicate progress when deleting, exporting, backing up, and restoring entries from Settings > Tools
  • Fixed a rare bug when displaying entries with calculated boluses.

v4.0 - October 20, 2012

  • Can choose how the app calculates insulin boluses. Choices are "insulin pump", which rounds to the nearest 0.05U and "MDI", which rounds to the nearest 0.5U. This is available under Settings > Boluses > Calculation type.
  • Can now set decimal values for carb bolus ratios.
  • Fixed crash when changing meal times on Android 2.1 (Eclair).

v3.0 - October 5, 2012

  • Can now export data as a CSV file from Settings > Tools.
  • Exported CSV file can be sent to others (via email) from within the app.
  • Edit screen now shows total insulin bolus.
  • Fixed bug when deleting entries and images on Android 4.1 (Jellybean).

v2.0 - September 19, 2012

  • Added the ability to record multiple types of insulin.
  • Can set a default insulin type for each meal.

v1.0 - September 9, 2012

  • First release!
